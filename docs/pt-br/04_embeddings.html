Embeddings

IDs de tokens sao apenas rotulos. O numero 32 nao significa nada por si so. Embeddings trocam cada ID por um vetor treinavel.

No inicio, esses vetores sao quase aleatorios. Durante o treino, eles sao ajustados para representar informacoes uteis para prever o proximo caractere.

Intuicao

Embedding e como transformar uma etiqueta simples em uma ficha com varias caracteristicas numericas que o modelo consegue combinar.
